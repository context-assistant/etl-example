# Markdown Documentation Example

This is a **Markdown** file demonstrating various formatting features commonly used in documentation, README files, and technical writing.

## Features

- **Bold text** for emphasis
- *Italic text* for subtle emphasis
- `Code snippets` for technical terms
- Lists for organizing information
